Classical measurement error

Population model: \(y^\ast = \alpha + \beta x^\ast + \varepsilon\)

Measurement error: \(\class{math-jax-red}{x} = \class{math-jax-blue}{x^\ast}+ e_x\), \(\class{math-jax-red}{y} = \class{math-jax-blue}{y^\ast}+ e_y\),

Observed model: \(y = \hat{\alpha} + \hat{\beta} x + \varepsilon\)
